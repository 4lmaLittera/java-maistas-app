# Maistas JavaFX Starter

Project bootstrapped with Maven to run a JavaFX desktop client for the maisto rezervavimo sistema kursinis.

## Prerequisites

- Java Development Kit 17 or newer (`java -version`)
- Apache Maven 3.9+

## Useful Commands

- `mvn clean javafx:run` – compile and launch the JavaFX application.
- `mvn test` – run unit tests (includes a basic JavaFX sanity check).

## Project Structure

- `src/main/java/com/naujokaitis/maistas/App.java` – primary JavaFX application entry point.
- `pom.xml` – Maven build configuration with OpenJFX dependencies.
